# テンプレート集

各カテゴリの初期ファイルおよびドキュメントテンプレート。

---

## inbox/README.md

```markdown
# Inbox — クイックキャプチャ

思いついたこと、気になったこと、後で整理したいことをここに記録します。

## 使い方

- `/secretary メモ [内容]` で即座にキャプチャ
- 週次レビュー時に他のカテゴリへ振り分ける

## ファイル命名規則

`YYYY-MM-DD.md` — 1日1ファイル
```

## reviews/README.md

```markdown
# Reviews — 週次・月次レビュー

定期的な振り返りを記録します。

## 使い方

- `/secretary 週次レビュー` で今週のレビューを自動生成
- 毎週金曜日または週末に実施推奨
```
